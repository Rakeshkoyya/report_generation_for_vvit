how to import student file to database

complete details on importing the students' data

You can import the student data from an excell file. The file should have the following requirements.

  • File format must be .xlsx
  • File should contain a column named "REGD NO"
  • File should contain a column named "FullName"
  • File should contain a column named "Branch"
  • Column headings must be exactly matched with above details.
  • Details must be in the first sheet (sheet1) of the file

NOTE: makesure your file containd all the students of the particular year. You can not edit the data in database so if you want to edit just delete the data from database and upload the edited file of students data (.xlsx file).

how this service works

The Report is generated by comparing the Attendance list file form MS teams,
This comparision can be done in two ways.
1.comparing with student data from our database.
2.comparing with student file given by you.

1.comparing with student data from our database.

Using student data from database

Attendance list form MS team by default is a .csv and we expect to receive a .csv for this field Sample Image of the Attendance list from MS teams is shown.

  • File format must be .csv
  • File should contain a column named "Full Name"
  • File should contain a column named "User Action"
  • File should contain a column named "Timestamp"
  • column headings must be exactly matched with above details

To use this service fill all the details in "Generate Your Attendance Report" form except the optional one, The program will decide to compare your Attendance list with our database.

NOTE: The default Attendance file downloaded from MS teams has all the above constraints. Do not upload the file which is edited

2.comparing with your given students data.

Using student data from your file

we except an excell file (.xlsx) as your students data. Sample Image of the student data is shown. The excell file must be formatted in the following way.

  • File format must be .xlsx
  • File should contain a column named "REGD NO"
  • File should contain a column named "FullName"
  • File should contain a column named "Branch"
  • Column headings must be exactly matched with above details.
  • Details must be in the first sheet (sheet1) of the file

To use this service fill all the details in "Generate Your Attendance Report" form including the optional one, The program will decide to compare your Attendance list with your file of students' data.
